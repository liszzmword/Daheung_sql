# 데이터 업로드 가이드

## 1. 세일즈 데이터 (CSV)

### 파일명 규칙
- 자유롭게 지정 가능
- 예: `sales_2025.csv`, `sales_2026_01.csv`, `1월매출.csv`
- **같은 파일명으로 재업로드하면 해당 파일 데이터만 교체됨** (다른 파일 데이터는 유지)

### 파일 형식
- 확장자: `.csv`
- 인코딩: UTF-8 (BOM 포함 가능)
- 첫 행: 컬럼명 (아래 순서)

### 필수 컬럼명
| 컬럼명 | 설명 | 예시 |
|--------|------|------|
| row_no | 행 번호 | 1 |
| sale_date | 매출일 | 2025.1.15 또는 2025-01-15 |
| customer_name | 거래처명 | 세명아크릴 |
| customer_code | 거래처 코드 | 20250115001 |
| sales_rep | 영업 담당자 | 김도순 |
| product_name | 제품명 | PS양면 |
| product_group | 제품군 | OTH |
| qty | 수량 | 2,000 |
| unit_price | 단가 | 2,060 |
| supply_amount | 공급가액 | 103,000 |
| margin_rate_pct | 마진율 | 20% |
| vat | 부가세 | 10,300 |
| total_amount | 합계 | 113,300 |

### 예시 (CSV)
```
row_no,sale_date,customer_name,customer_code,sales_rep,product_name,product_group,qty,unit_price,supply_amount,margin_rate_pct,vat,total_amount
1,2026.1.5,세명아크릴,20260105001,김도순,PS양면,OTH,"2,000","2,060","103,000",20%,"10,300","113,300"
```

## 2. 영업일지 (XLS/XLSX)

### 파일명 규칙 (중요!)
- **반드시 파일명에 4자리 연도를 포함**
- 형식: `sales_diary_YYYY_MM.xls`
- 예:
  - `sales_diary_2025_12.xls` → 2025년 12월
  - `sales_diary_2026_01.xls` → 2026년 1월
  - `sales_diary_2026_02.xlsx` → 2026년 2월
- **같은 파일명으로 재업로드하면 해당 파일 데이터만 교체됨**

### 파일 구조
- 확장자: `.xls` 또는 `.xlsx`
- **시트 이름 = 영업사원 이름** (예: 김한섭, 박세용)
- 모든 시트는 동일한 7개 컬럼 구조

### 컬럼 구조 (7개, 순서대로)
| 순서 | 내용 | 예시 |
|------|------|------|
| 1열 | 날짜 | 12월02일 |
| 2열 | 기업 | 현대자동차 |
| 3열 | 기업담당자 | 김부장 |
| 4열 | 시작시간 | 09:00 |
| 5열 | 끝시간 | 10:30 |
| 6열 | 방문/제품명 | 방문 |
| 7열 | 내용 | 신규 제품 샘플 전달 |

### 날짜 작성 규칙
- 한국어 형식: `12월02일`, `1월15일`
- 날짜는 첫 행에만 쓰면 됨 (같은 날짜의 다른 기업은 날짜 칸 비워도 됨)
- 연도는 파일명에서 자동 추출됨

### 예시 (시트: 김한섭)
| 날짜 | 기업 | 기업담당자 | 시작시간 | 끝시간 | 방문/제품명 | 내용 |
|------|------|-----------|---------|--------|-----------|------|
| 12월02일 | 현대자동차 | 김부장 | 09:00 | 10:30 | 방문 | 신규 제품 샘플 전달 |
| | 삼성전자 | 이과장 | 14:00 | 15:00 | PS양면 | 견적 요청 받음 |
| 12월03일 | LG전자 | 박대리 | 10:00 | 11:00 | 방문 | 계약 조건 협의 |

## 3. 업로드 방법

1. 웹사이트 로그인
2. 상단 **"업로드"** 버튼 클릭
3. 파일 드래그 또는 클릭하여 선택
4. 유형 자동 감지 (CSV → 세일즈, XLS → 영업일지)
5. 업로드 완료 시 처리 건수 표시

---

## 4. 주의사항

- 파일 크기: 최대 10MB
- 같은 파일명으로 재업로드하면 해당 파일의 기존 데이터만 교체 (다른 파일 데이터는 그대로 유지)
- 세일즈 CSV의 숫자에 콤마(,)가 있어도 자동 처리됨
- 영업일지의 날짜에 연도를 쓸 필요 없음 (파일명에서 자동 추출)
